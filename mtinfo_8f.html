subroutine mtinfo ( character*(*)  CMTDIR,
  LUNMT1,
  LUNMT2 
)

THIS SUBROUTINE IS USED TO SPECIFY THE DIRECTORY LOCATION AND FORTRAN LOGICAL UNIT NUMBERS TO USE WHEN READING BUFR MASTER TABLES ON THE LOCAL FILE SYSTEM.

THE INPUT LOGICAL UNIT NUMBERS SHOULD BE UNIQUE BUT SHOULD NOT ALREADY BE ASSIGNED TO ANY ACTUAL BUFR MASTER TABLE FILES. IF THIS SUBROUTINE IS NOT CALLED, THEN DEFAULT VALUES ARE USED AS DEFINED WITHIN BUFR ARCHIVE LIBRARY SUBROUTINE BFRINI.

PROGRAM HISTORY LOG: 2009-03-23 J. ATOR – ORIGINAL AUTHOR

USAGE: CALL MTINFO ( CMTDIR, LUNMT1, LUNMT2 ) INPUT ARGUMENT LIST: CMTDIR - CHARACTER*(*): DIRECTORY LOCATION OF BUFR MASTER TABLES ON LOCAL FILE SYSTEM (UP TO 100 CHARACTERS) LUNMT1 - INTEGER: FIRST FORTRAN LOGICAL UNIT NUMBER TO USE WHEN READING BUFR MASTER TABLES ON LOCAL FILE SYSTEM LUNMT2 - INTEGER: SECOND FORTRAN LOGICAL UNIT NUMBER TO USE WHEN READING BUFR MASTER TABLES ON LOCAL FILE SYSTEM

REMARKS: THIS ROUTINE CALLS: BORT2 STRSUC THIS ROUTINE IS CALLED BY: None Normally called only by application programs.
