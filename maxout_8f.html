subroutine maxout (   MAXO)

THIS SUBROUTINE ALLOWS AN APPLICATION PROGRAM TO SET THE RECORD LENGTH OF NEWLY CREATED BUFR MESSAGES, OVERRIDING THE VALUE SET IN BUFR ARCHIVE LIBRARY SUBROUTINE BFRINI.

THIS MUST BE CALLED AFTER THE INITIAL CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE OPENBF SINCE OPENBF CALLS BFRINI. THE RECORD LENGTH WILL REMAIN MAXO UNLESS THIS SUBROUTINE IS CALLED AGAIN WITH A NEW MAXO.
