subroutine copymg (   LUNIN,
  LUNOT 
)

THIS SUBROUTINE COPIES A BUFR MESSAGE, INTACT, FROM LOGICAL UNIT LUNIN, OPENED FOR INPUT VIA A PREVIOUS CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE OPENBF, TO LOGICAL UNIT LUNOT, OPENED FOR OUTPUT VIA A PREVIOUS CALL TO OPENBF.

THE MESSAGE COPIED FROM LOGICAL UNIT LUNIN WILL BE THE ONE MOST RECENTLY READ USING BUFR ARCHIVE LIBRARY SUBROUTINE READMG. THE OUTPUT FILE MUST HAVE NO CURRENTLY OPEN MESSAGES. ALSO, BOTH FILES MUST HAVE BEEN OPENED TO THE BUFR INTERFACE WITH IDENTICAL BUFR TABLES.

USAGE: CALL COPYMG (LUNIN, LUNOT) INPUT ARGUMENT LIST: LUNIN - INTEGER: FORTRAN LOGICAL UNIT NUMBER FOR INPUT BUFR FILE LUNOT - INTEGER: FORTRAN LOGICAL UNIT NUMBER FOR OUTPUT BUFR FILE
