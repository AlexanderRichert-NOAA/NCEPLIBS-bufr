function idxmsg ( dimension(*)  MESG)

THIS FUNCTION DETERMINES WHETHER THE GIVEN BUFR MESSAGE IS A DX DICTIONARY MESSAGE THAT WAS CREATED BY THE BUFR ARCHIVE LIBRARY SOFTWARE.

PROGRAM HISTORY LOG: 2009-03-23 J. ATOR – ORIGINAL AUTHOR

USAGE: IDXMSG( MESG ) INPUT ARGUMENT LIST: MESG - INTEGER: *-WORD PACKED BINARY ARRAY CONTAINING BUFR MESSAGE

OUTPUT ARGUMENT LIST: IDXMSG - INTEGER: RETURN VALUE: 0 - MESG IS NOT A DX DICTIONARY MESSAGE 1 - MESG IS A DX DICTIONARY MESSAGE

REMARKS: THIS ROUTINE CALLS: IUPBS01 THIS ROUTINE IS CALLED BY: CPDXMM DATEBF DUMPBF MESGBC MESGBF MSGWRT RDBFDX READMG POSAPX READERME UFBMEM Normally not called by application programs but it could be.
