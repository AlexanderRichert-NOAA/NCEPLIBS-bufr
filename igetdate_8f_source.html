4 C> THIS FUNCTION UNPACKS AND RETURNS THE SECTION 1 DATE-TIME
5 C> FROM THE BUFR MESSAGE STORED IN ARRAY MBAY. IT WILL WORK ON ANY
6 C> MESSAGE ENCODED USING BUFR EDITION 2, 3 OR 4. THE START OF THE
7 C> BUFR MESSAGE, (I.E. THE STRING "BUFR") MUST BE ALIGNED ON THE FIRST
8 C> FOUR BYTES OF MBAY.

16 C> BUFR MESSAGE
17 C>
18 C> OUTPUT ARGUMENT LIST:

21 C> IMO - INTEGER: SECTION 1 MONTH (MM)
22 C> IDY - INTEGER: SECTION 1 DAY (DD)
