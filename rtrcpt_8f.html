THIS SUBROUTINE RETURNS THE TANK RECEIPT TIME STORED WITHIN SECTION 1 OF THE BUFR MESSAGE OPEN FOR INPUT VIA A PREVIOUS CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE READMG, READMM OR EQUIVALENT.

PROGRAM HISTORY LOG: 2009-03-23 J. ATOR – ORIGINAL AUTHOR 2013-10-07 J. ATOR – MODIFIED TO CALL RTRCPTB 2014-12-10 J. ATOR – USE MODULES INSTEAD OF COMMON BLOCKS

USAGE: CALL RTRCPT (LUNIT,IYR,IMO,IDY,IHR,IMI,IRET) INPUT ARGUMENT LIST: LUNIT - INTEGER: FORTRAN LOGICAL UNIT NUMBER FOR BUFR FILE

OUTPUT ARGUMENT LIST: IYR - INTEGER: TANK RECEIPT YEAR IMO - INTEGER: TANK RECEIPT MONTH IDY - INTEGER: TANK RECEIPT DAY IHR - INTEGER: TANK RECEIPT HOUR IMI - INTEGER: TANK RECEIPT MINUTE IRET - INTEGER: RETURN CODE: 0 = normal return -1 = no tank receipt time was present within the BUFR message currently open for input

REMARKS: THIS ROUTINE CALLS: BORT RTRCPTB STATUS THIS ROUTINE IS CALLED BY: None Normally called only by application programs.
