function iupbs3 ( dimension(*)  MBAY,
character*(*)  S3MNEM 
)

THIS FUNCTION UNPACKS AND RETURNS A SPECIFIED INTEGER VALUE FROM SECTION 3 OF THE BUFR MESSAGE STORED IN ARRAY MBAY.

IT WILL WORK ON ANY MESSAGE ENCODED USING BUFR EDITION 2, 3 OR 4. THE START OF THE BUFR MESSAGE (I.E. THE STRING "BUFR") MUST BE ALIGNED ON THE FIRST FOUR BYTES OF MBAY, AND THE VALUE TO BE UNPACKED IS SPECIFIED VIA THE MNEMONIC S3MNEM, AS EXPLAINED IN FURTHER DETAIL BELOW.

PROGRAM HISTORY LOG: 2009-03-23 J. ATOR – ORIGINAL AUTHOR

USAGE: IUPBS3 (MBAY, S3MNEM) INPUT ARGUMENT LIST: MBAY - INTEGER: -WORD PACKED BINARY ARRAY CONTAINING BUFR MESSAGE S3MNEM - CHARACTER(*): MNEMONIC SPECIFYING VALUE TO BE UNPACKED FROM SECTION 3 OF BUFR MESSAGE: 'NSUB' = NUMBER OF DATA SUBSETS 'IOBS' = FLAG INDICATING WHETHER THE MESSAGE CONTAINS OBSERVED DATA: 0 = NO 1 = YES 'ICMP' = FLAG INDICATING WHETHER THE MESSAGE CONTAINS COMPRESSED DATA: 0 = NO 1 = YES

OUTPUT ARGUMENT LIST: IUPBS3 - INTEGER: UNPACKED INTEGER VALUE -1 = THE INPUT S3MNEM MNEMONIC WAS INVALID

REMARKS: THIS ROUTINE CALLS: GETLENS IUPB THIS ROUTINE IS CALLED BY: CKTABA CPDXMM DUMPBF MESGBC RDBFDX READERME STNDRD WRITLC Also called by application programs.
