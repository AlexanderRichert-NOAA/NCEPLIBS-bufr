4 C> THIS FUNCTION SHOULD ONLY BE CALLED IF DYNAMIC MEMORY
5 C> ALLOCATION IS BEING USED. IT SETS A SPECIFIED PARAMETER TO A
6 C> SPECIFIED VALUE FOR LATER USE IN DYNAMICALLY SIZING ONE OR MORE
7 C> INTERNAL ARRAYS. THE LIST OF PARAMETERS THAT CAN BE DYNAMICALLY
8 C> SIZED IS GIVEN BELOW, AND A SEPARATE CALL TO THIS FUNCTION MUST
9 C> BE MADE FOR EACH SUCH PARAMETER THAT IS DESIRED TO BE CHANGED
10 C> FROM ITS DEFAULT VALUE. IF THIS FUNCTION IS NOT CALLED FOR A
11 C> PARTICULAR PARAMETER, THE DEFAULT VALUE IS OBTAINED FROM THE
12 C> CORRESPONDING MODULE FILE FOR THAT PARAMETER.
13 C>
14 C> NOTE THAT ALL CALLS TO THIS FUNCTION MUST BE MADE PRIOR TO THE
15 C> FIRST CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE OPENBF.
16 C>
