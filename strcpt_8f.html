THIS SUBROUTINE CAN BE CALLED AT ANY TIME AFTER THE FIRST CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE OPENBF.

WHEN CF IS SET TO 'Y' (= 'YES'), THIS SUBROUTINE IS USED TO SPECIFY A TANK RECEIPT TIME THAT WILL BE APPENDED TO SECTION 1 OF ALL FUTURE BUFR MESSAGES OUTPUT BY ANY OF THE BUFR ARCHIVE LIBRARY SUBROUTINES WHICH WRITE SUCH MESSAGES (E.G. WRITSB, COPYMG, WRITSA, ETC.). WHEN CF IS SET TO 'N' (= 'NO', WHICH IS THE DEFAULT), THIS CAPABILITY IS TURNED OFF (IF IT WAS PREVIOUSLY TURNED ON) AND THE VALUES IN ALL OF THE OTHER INPUT ARGUMENTS ARE IGNORED. THE TANK RECEIPT TIME IS A LOCAL EXTENSION TO SECTION 1; HOWEVER, ITS INCLUSION IN A MESSAGE IS STILL FULLY COMPLIANT WITH THE WMO FM-94 BUFR REGULATIONS.

PROGRAM HISTORY LOG: 2009-03-23 J. ATOR – ORIGINAL AUTHOR

USAGE: CALL STRCPT (CF,IYR,IMO,IDY,IHR,IMI) INPUT ARGUMENT LIST: CF - CHARACTER*1: FLAG INDICATING WHETHER FUTURE CALLS TO BUFRLIB MESSAGE WRITING ROUTINES (E.G. WRITSB, COPYMG, WRITSA, ETC.) SHOULD APPEND THE GIVEN TANK RECEIPT TIME TO SECTION 1 OF SUCH MESSAGES: 'N' = 'NO' (THE DEFAULT) 'Y' = 'YES' IYR - INTEGER: TANK RECEIPT YEAR TO BE STORED IMO - INTEGER: TANK RECEIPT MONTH TO BE STORED IDY - INTEGER: TANK RECEIPT DAY TO BE STORED IHR - INTEGER: TANK RECEIPT HOUR TO BE STORED IMI - INTEGER: TANK RECEIPT MINUTE TO BE STORED

REMARKS: THIS ROUTINE CALLS: BORT CAPIT THIS ROUTINE IS CALLED BY: None Normally called only by application programs.
