subroutine wrdlen

THIS SUBROUTINE FIGURES OUT SOME IMPORTANT INFORMATION ABOUT THE LOCAL MACHINE ON WHICH THE BUFR ARCHIVE LIBRARY SOFTWARE IS BEING RUN AND STORES THIS INTO COMMON BLOCK /HRDWRD/.

SUCH INFORMATION INCLUDES DETERMINING THE NUMBER OF BITS AND THE NUMBER OF BYTES IN A MACHINE WORD AS WELL AS DETERMINING WHETHER THE MACHINE USES THE ASCII OR EBCDIC CHARACTER SET.

NOTE: IT IS ONLY NECESSARY FOR THIS SUBROUTINE TO BE CALLED ONCE, AND THIS IS NORMALLY DONE DURING THE FIRST CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE OPENBF. HOWEVER, THE SUBROUTINE DOES KEEP TRACK OF WHETHER IT HAS ALREADY BEEN CALLED; THUS, IF IT IS CALLED AGAIN LATER BY A DIFFERENT BUFR ARCHIVE LIBRARY SUBROUTINE, IT WILL JUST QUIETLY RETURN WITHOUT (RE)COMPUTING ALL OF THE INFORMATION WITHIN COMMON BLOCK /HRDWRD/.
