GIVEN A MNEMONIC WITHIN A SUBSET CURRENTLY OPEN FOR READING, THIS SUBROUTINE CHECKS WHETHER THE MNEMONIC REFERENCES ANOTHER ELEMENT WITHIN THE SAME SUBSET VIA AN INTERNAL BITMAP, AND IF SO RETURNS THE REFERENCED MNEMONIC AND ITS LOCATION.

IF THERE IS MORE THAN ONE OCCURRENCE OF THE INPUT MNEMONIC WITHIN THE OVERALL SUBSET DEFINITION, THE SUBROUTINE WILL RETURN THE MNEMONIC REFERRED TO BY THE (NTAGI)th OCCURRENCE OF THE INPUT MNEMONIC, COUNTING FROM THE BEGINNING OF THE OVERALL SUBSET DEFINITION. A SUBSET DEFINITION MUST ALREADY BE IN SCOPE, EITHER VIA A PREVIOUS CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE READSB OR EQUIVALENT.

PROGRAM HISTORY LOG: 2016-06-07 J. ATOR – ORIGINAL AUTHOR

USAGE: CALL GETTAGRE (LUNIT, TAGI, NTAGI, TAGRE, NTAGRE, IRET) INPUT ARGUMENT LIST: LUNIT - INTEGER: FORTRAN LOGICAL UNIT NUMBER FOR BUFR FILE TAGI - CHARACTER*(*): MNEMONIC NTAGI - INTEGER: ORDINAL OCCURRENCE OF TAGI FOR WHICH TAGRE IS TO BE RETURNED, COUNTING FROM THE BEGINNING OF THE OVERALL SUBSET DEFINITION

OUTPUT ARGUMENT LIST: TAGRE - CHARACTER*(*): MNEMONIC REFERRED TO BY TAGI VIA AN INTERNAL BITMAP NTAGRE - INTEGER: ORDINAL OCCURRENCE OF TAGRE REFERRED TO BY (NTAGI)th OCCURRENCE OF TAGI, COUNTING FROM THE BEGINNING OF THE OVERALL SUBSET DEFINITIION. IRET - INTEGER: RETURN CODE 0 = NORMAL RETURN -1 = COULD NOT FIND TAGRE

REMARKS: THIS ROUTINE CALLS: FSTAG STATUS STRSUC THIS ROUTINE IS CALLED BY: None Normally called only by application programs
