GIVEN A TABLE B MNEMONIC, THIS SUBROUTINE RETURNS THE SCALE FACTOR, REFERENCE VALUE AND BIT WIDTH CORRESPONDING TO THE (NNEMO)th OCCURRENCE OF THAT MNEMONIC WITHIN A SUBSET DEFINITION (COUNTING FROM THE BEGINNING OF THE OVERALL SUBSET DEFINITION), AND INCLUDING ACCOUNTING FOR ANY TABLE C OPERATORS (E.G.

2-01-YYY, 2-02-YYY, 2-03-YYY, 2-07-YYY) WHICH MAY BE IN EFFECT FOR THAT PARTICULAR OCCURRENCE OF THE MNEMONIC. A SUBSET DEFINITION MUST ALREADY BE IN SCOPE, EITHER VIA A PREVIOUS CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE READSB OR EQUIVALENT (FOR INPUT FILES) OR TO SUBROUTINE OPENMB OR EQUIVALENT (FOR OUTPUT FILES).

PROGRAM HISTORY LOG: 2014-10-02 J. ATOR – ORIGINAL VERSION 2014-12-10 J. ATOR – USE MODULES INSTEAD OF COMMON BLOCKS

USAGE: CALL NEMSPECS (LUNIT, NEMO, NNEMO, NSCL, NREF, NBTS, IRET ) INPUT ARGUMENT LIST: LUNIT - INTEGER: FORTRAN LOGICAL UNIT NUMBER FOR BUFR FILE NEMO - CHARACTER*(*): TABLE B MNEMONIC NNEMO - INTEGER: ORDINAL OCCURRENCE OF NEMO FOR WHICH INFORMATION IS TO BE RETURNED, COUNTING FROM THE BEGINNING OF THE OVERALL SUBSET DEFINITION

OUTPUT ARGUMENT LIST: NSCL - INTEGER: SCALE FACTOR CORRESPONDING TO NEMO NREF - INTEGER: REFERENCE VALUE CORRESPONDING TO NEMO NBTS - INTEGER: BIT WIDTH CORRESPONDING TO NEMO IRET - INTEGER: RETURN CODE 0 = NORMAL RETURN -1 = REQUESTED MNEMONIC COULD NOT BE FOUND, OR SOME OTHER ERROR OCCURRED

REMARKS: THIS ROUTINE CALLS: FSTAG STATUS STRSUC THIS ROUTINE IS CALLED BY: None Normally called only by application programs
