character*8  SUBSET,
  JDATE,
  IRET

THIS SUBROUTINE READS THE NEXT BUFR MESSAGE FROM LOGICAL UNIT NUMBER ABS(LUNXX) INTO AN INTERNAL MESSAGE BUFFER (I.E.

ARRAY MBAY IN MODULE BITBUF). ABS(LUNXX) SHOULD ALREADY BE OPENED FOR INPUT OPERATIONS. IF LUNXX < 0, THEN A READ ERROR FROM ABS(LUNXX) IS TREATED THE SAME AS THE END-OF-FILE (EOF) CONDITION; OTHERWISE, BUFR ARCHIVE LIBRARY SUBROUTINE BORT IS NORMALLY CALLED IN SUCH SITUATIONS. ANY DX DICTIONARY MESSAGES ENCOUNTERED WITHIN ABS(LUNXX) ARE AUTOMATICALLY PROCESSED AND STORED INTERNALLY, SO A SUCCESSFUL RETURN FROM THIS SUBROUTINE WILL ALWAYS RESULT IN A BUFR MESSAGE CONTAINING ACTUAL DATA VALUES.

USAGE: CALL READMG (LUNXX, SUBSET, JDATE, IRET) INPUT ARGUMENT LIST: LUNXX - INTEGER: ABSOLUTE VALUE IS FORTRAN LOGICAL UNIT NUMBER FOR BUFR FILE (IF LUNXX IS LESS THAN ZERO, THEN READ ERRORS FROM ABS(LUNXX) ARE TREATED THE SAME AS EOF)

OUTPUT ARGUMENT LIST: SUBSET - CHARACTER*8: TABLE A MNEMONIC FOR TYPE OF BUFR MESSAGE BEING READ JDATE - INTEGER: DATE-TIME STORED WITHIN SECTION 1 OF BUFR MESSAGE BEING READ, IN FORMAT OF EITHER YYMMDDHH OR YYYYMMDDHH, DEPENDING ON DATELEN() VALUE IRET - INTEGER: RETURN CODE: 0 = normal return -1 = there are no more BUFR mesages in ABS(LUNXX)

REMARKS: THIS ROUTINE CALLS: BORT CKTABA ERRWRT IDXMSG RDBFDX RDMSGW READS3 STATUS WTSTAT BACKBUFR THIS ROUTINE IS CALLED BY: IREADMG READNS RDMGSB REWNBF UFBINX UFBPOS Also called by application programs.
